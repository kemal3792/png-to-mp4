# png-to-mp4
This script is designed to turn all pngs in a folder and its subfolders into a video by adding a canvas background.

some background color codes:
White: (255, 255, 255)
Black: (0, 0, 0)
Red: (255, 0, 0)
Green: (0, 255, 0)
Blue: (0, 0, 255)
Light Blue: (173, 216, 230)
Light Green: (144, 238, 144)
